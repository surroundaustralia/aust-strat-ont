Description

This is an extremely simple ontology - part of the Stratigraphic Units profile - whos purpose is only to provide a Semantic Web view of the data contained within the Australian Stratigraphic Units Database. Data from that database is also available according to a number of other profiles which, in some cases, this ontology specialises.

Is Defined By https://linked.data.gov.au/def/su/ont
Source http://www.opengis.net/gsml/4.1/GeoSciML-Basic/GeologicUnit
Description

A temporal extent during which one or more geological processes acted to modify geological entities such as Stratigraphic Units. A Geologic Event has a specified geologic age and may have specified environments and processes.

Stratigraphic Unitc

URI https://linked.data.gov.au/def/su/StratigraphicUnit
Is Defined By https://linked.data.gov.au/def/su/ont
Source http://www.opengis.net/gsml/4.1/GeoSciML-Basic/GeologicUnit
Description

A Stratigraphic Unit is a volume of rock of identifiable origin and relative age range that is defined by the distinctive and dominant, easily mapped and recognizable petrographic, lithologic or paleontologic features (facies) that characterise it.
Super-classes geo:Featurec
In domain of su:geologicHistoryop
